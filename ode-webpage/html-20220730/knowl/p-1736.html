The differential equation

\begin{equation*} y'' + 6y' + 25y = 0, \quad y(0) = 1, \quad y'(0) = 0 \end{equation*}

has solution

\begin{equation} y(t) = \frac{1}{4} e^{-3t} (3 \sin(4t) + 4 \cos(4t)).\tag{4.5.8} \end{equation}

The phase shifted form of (4.5.8) is

\begin{align*} y(t) \amp = \frac{5}{4} e^{-3t} \sin\left( 4t + \arctan\left( \frac{4}{3} \right) \right)\\ \amp = \frac{5}{4} e^{-3t} \sin\left( 4 \left( t + \frac{(\arctan(4/3)}{4} \right) \right)\\ \amp = \frac{5}{4} e^{-3t} \sin(4 ( t + 0.231824 ) ), \end{align*}

where \(\omega =4\) and \(\theta = \arctan(4/3) = 0.231824\) radians.
