Conversely, assume that

If we differentiate both sides of this equation, we obtain \(u'(t) = f(t, u(t))\text{.}\) Since

\begin{equation*} u(t_0) = x_0 + \int_{t_0}^{t_0} f(s, u(s)) \, ds = x_0, \end{equation*}
