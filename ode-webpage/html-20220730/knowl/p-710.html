We can use direction fields in the phase plane to represent autonomous systems

\begin{align} \frac{dx}{dt} & = f(x, y),\tag{2.2.1}\\ \frac{dy}{dt} & = g(x, y).\tag{2.2.2} \end{align}

Equation (2.2.1) tells us how a solution curve changes in the \(x\) direction, while equation (2.2.2) tells us how a solution curve changes in the \(y\) direction.
