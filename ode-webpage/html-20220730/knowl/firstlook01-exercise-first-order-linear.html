Exercise 29.

\begin{equation*} x(t) = Ce^{-at} + e^{-at} \int_{t_0}^t e^{as}q(s) \, ds \end{equation*}

is a solution of this equation, where \(c\) is any constant and \(t_0 \in I\text{.}\)
